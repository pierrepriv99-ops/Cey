## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/pierrepriv99-ops/Cey.git

# Install
npm install

# Compile contracts
npx hardhat compile

# Test
npx hardhat test
```

## 📦 Modules

- `/contracts` - Smart contracts (Solidity)
- `/mobile` - Android apps
- `/desktop` - Electron desktop
- `/network` - P2P mesh
- `/mind` - AI shell
- `/website` - Next.js website
- `/sdk` - TypeScript SDK
